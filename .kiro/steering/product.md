# Product Overview

## Grammar Proofreading Extension

A VSCode extension that provides AI-powered grammar correction and proofreading capabilities using Large Language Models (LLMs). The extension integrates with OpenAI-compatible APIs to analyze and improve text quality through predefined correction types.

### Core Features

- **Multiple Correction Types**: Grammar, style, clarity, tone, and custom corrections
- **LLM Integration**: Works with OpenAI-compatible APIs (OpenAI, Azure OpenAI, local servers)
- **VSCode Integration**: Context menu commands, command palette, and editor integration
- **User Experience**: Progress indicators, detailed explanations, undo support
- **Flexible Configuration**: Configurable API endpoints, models, and parameters

### Target Users

Developers, writers, and content creators who want to improve their text quality directly within VSCode using AI assistance.

### Key Value Proposition

Seamless integration of AI-powered text correction capabilities directly into the VSCode editor workflow, supporting multiple correction types and flexible API configurations.
